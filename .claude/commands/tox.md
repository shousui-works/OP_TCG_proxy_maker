# tox実行と修正

`uv run tox` を実行して、テストとlintがすべて通るように修正してください。

## 手順

1. `uv run tox` を実行する
2. 失敗したテストやlintエラーがあれば、その内容を確認する
3. エラーの原因を特定し、コードを修正する
4. 再度 `uv run tox` を実行して、すべてパスすることを確認する
5. すべてパスするまで手順2-4を繰り返す

## 注意事項

- テストコードに問題がある場合は、テストコードを修正する
- lintエラー（ruff）がある場合は、コードスタイルを修正する
- 必要に応じて `uv run ruff check --fix .` で自動修正を試みる
